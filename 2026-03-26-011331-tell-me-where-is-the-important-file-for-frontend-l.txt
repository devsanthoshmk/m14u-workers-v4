     Plan: Replace M14U-js Backend & Logic with ytify Architecture

     Context

     M14U-js currently uses Express + ytmusic-api + yt-dlp for music fetching.
     The goal is to replace this with ytify's architecture: Cloudflare Workers +
      youtubei.js backend, Invidious proxies for audio streams, and Netlify edge
      functions as fallback. The Listen Along feature will be disabled with a
     "Coming Soon" tag. The React 19 UI stays but gets rewired to the new API
     layer.

     matching ytify:
